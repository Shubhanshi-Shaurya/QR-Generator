## QR Generator

A simple **QR Generator** built using **HTML, CSS, and JavaScript**. A clean and minimal QR Generator built using basic web technologies. This project generate QR codes from text or URLs instantly.

---

## Features
* **Simple UI:** Clean and easy to use.
* **Instant:** Generates as you type

---

## Technologies Used
- **HTML** – Page structure
- **CSS** – Styling and layout
- **JavaScript** – Logic for API
